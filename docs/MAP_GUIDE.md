## 🎮 특수 시스템 정리

### 시간 시스템 (timeSystem.js)
- 게임 1일 = 현실 12분
- 밤(20:00~06:00): 강력 몬스터 출현 확률↑
- 자연 재생: 4초당 HP/MP 1% 회복 (전투 중 X)

### 맵 연결 순서
```
훈련장 → 버려진 마을 → 수상한 동굴 → 이상한 숲 → 고대의 탑 → 드래곤 레어 → 악마의 성
```
