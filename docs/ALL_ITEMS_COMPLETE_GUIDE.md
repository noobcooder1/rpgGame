### 2.4 부분 구현 아이템

완전히 미구현은 아니지만 일부 기능이 동작하지 않는 아이템.

#### 정화의 물약 (purify_potion)

- **등급**: Uncommon
- **구매가**: 25G / **판매가**: 12G
- **획득처**: 상점 구매
- **상태**: ⚠️ 부분 구현
- **문제**: `shopSystem.js`에 `{ type: 'cure_status' }` 효과 정의는 있으나, `inventorySystem.js`의 `useItem()` 함수에서 `cure_status` 처리 로직이 없어 사용해도 **아무 효과가 없음**
- **의도된 효과**: 독, 화상 등 모든 상태이상 제거

#### 힘의 엘릭서 (strength_elixir)

- **등급**: Epic
- **획득처**: 동굴 트롤 드롭 (60%)
- **판매가**: 250G
- **상태**: ⚠️ 부분 구현
- **문제**: 아이템 설명에는 "힘"과 관련이 있지만 실제 효과는 `{ hp: 100 }` (HP 회복)만 구현됨. 공격력 강화 등의 추가 효과 없음

#### 알 수 없는 책 (unknown_book)

- **등급**: Epic
- **획득처**: 저주받은 영주 드롭 (70%)
- **판매가**: 500G
- **상태**: ⚠️ 효과 미구현
- **문제**: 특수(special) 타입으로 장착 불가. 설명 외에 실제 사용 효과 없음. 스킬 해금 등의 기능 미구현

#### 고대 룬 (ancient_rune)

- **등급**: Legendary
- **획득처**: 고대 수호자 드롭 (100%)
- **판매가**: 1500G
- **상태**: ⚠️ 효과 미구현
- **문제**: 특수(special) 타입으로 장착 불가. "고대의 마력이 깃든 신비한 룬"이라는 설명만 있고 실제 기능 없음

## 3. 아이템 시스템 구조

### 희귀도 (Rarity) 등급

| 등급 | 색상 | 설명 |
|------|------|------|
| Common (일반) | 회색 `#9e9e9e` | 초기 장비, 기본 소모품 |
| Uncommon (고급) | 초록 `#4caf50` | 상점 판매 장비, 일부 드롭 |
| Rare (희귀) | 파랑 `#2196f3` | 대련 보상, 저주 장비, 보스 드롭 |
| Epic (영웅) | 보라 `#9c27b0` | 특수 아이템, 엘릭서, 마법 장비 |
| Legendary (전설) | 주황 `#ff9800` | 최종 보스 드롭 한정 |

### 장비 슬롯 종류

| 타입 | 슬롯 | 비고 |
|------|------|------|
| `weapon` | 무기 | 직업 제한 있음 |
| `armor` | 방어구 | 직업 제한 있음 |
| `helmet` | 투구 | 현재 구현된 아이템 없음 |
| `boots` | 부츠 | 저주받은 신발, 신속의 부츠 |
| `gloves` | 장갑 | 저주받은 장갑, 마법 장갑 등 |
| `necklace` | 목걸이 | 뼈 목걸이, 왕의 목걸이 등 |
| `ring` | 반지 | 뼈 반지, 왕의 반지 등 |
| `accessory` | 악세서리 | 탐사 아이템 |
| `tool` | 도구 | 현재 구현된 아이템 없음 |

### 아이템 획득 경로 요약

| 획득 방법 | 아이템 종류 |
|-----------|-------------|
| 게임 시작 지급 | 직업별 초기 장비 1세트, HP포션 5개, MP포션 3개 |

| 대련 승리 | 강철 무기 4종 (직업별 1개) |
| 탐사 | 조잡한 아이템, 골드, 경험치 |
| 퀘스트 보상 | 현재 없음 (동굴트롤 이빨은 퀘스트 제출 아이템) |
